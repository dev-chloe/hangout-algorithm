## 유클리드 호제법

유클리드 호제법(또는 유클리드 알고리즘)은 2**개의 자연수 또는 정식의 최대 공약수를 구하는 알고리즘**의 하나이다.  
호제법은 두 수가 서로 상대수를 나누어 결국 원하는 수를 얻는 알고리즘을 말한다.  

ex) 722와 190의 최대 공약수

```javascript
722 / 190 = 3  // 나머지 152
190 / 152 = 1  // 나머지 38
152 / 38 = 4 // 나머지 0

// 722와 190의 최대 공약수는 38이 최대공약수이다.
```
